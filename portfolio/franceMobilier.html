Projet

Ce projet consiste en la création d'un site web en PHP permettant de répertorier des meubles en fonction d'une gamme et d'une catégorie, Ce projet est réalisé sous l'architecture MVC (Modele - Vue - Controleur) . Les informations sur les meubles et les clients sont stockées dans une base de donnée MySql.

Fonctionnement du site

Dans ce projet le client peut sélectionner une catégorie ou une gamme de meubles la priorité de recherche est donné à la catégorie puis la gamme, il peut donc voir le nom du meuble, sa photo ainsi que son prix en rapport avec le filtre choisi.

Inscription à une newsletter

Le site permet aussi de s'inscrire à une newsletter en sélectionnant un type précis de gamme pour être informé au moindre nouvel arrivage de cette gamme

Le client reçoit ensuite un mail de confirmation et est répertorié dans la base de données.
