Docker

Docker est un logiciel permettant de créer, déployer et éxecuter des applications dans des contenurs. Un conteneur est une unité logicielle permettant d'emballer une application et ses dépendances, il contient donc toutes les fichiers de configuration, ainsi que les bibliothéques nécessaires au bon fonctionnement de l'application

Pourquoi utiliser Docker?

Docker a une particularité très importante étant le fait que leurs conteneurs soient portables c'est à dire que peu importe l'environnement d'éxecution de l'application celle ci fonctionnera et permet de faciliter la migration entre différents environnements. Docker lors du lancement de l'application crée un environnement isolé du reste et donc fonctionne indépendamment des autres applications sur la machine. Ainsi, on peut utiliser l'application peu importe l'environnement.

En résumé Docker est une excellente solution pour tout développeur voulant créer et gérer une application de manière efficace avec un environnement portable, isolé, et fonctionnant partout.
